# AI Climate Digital Twin - India 🇮🇳

An interactive Machine Learning powered dashboard for predicting and visualizing rainfall patterns across India.

Coming Soon

## 📌 Overview

This project is an early prototype of an **AI Climate Digital Twin for India**.

The dashboard allows users to:

- View actual rainfall data
- Explore predicted rainfall values
- Visualize prediction errors
- Categorize rainfall risk
- Simulate climate scenarios using interactive controls
- Explore rainfall patterns across states and districts

The goal is to build an intelligent climate decision-support system that can eventually assist in:

- Agriculture planning
- Flood preparedness
- Water resource management
- Climate risk analysis

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/AI-Climate-Digital-Twin.git
```

Move into the folder:

```bash
cd AI-Climate-Digital-Twin
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the app:

```bash
streamlit run app.py
```
